About BlameStella

BlameStella.com is a web monitoring service built with Ruby, Redis, PostgreSQL, and PhantomJS. It checks your site every few minutes and notifies you when there's a problem. Since we render entire pages we can tell when there's problems with your javascript too.

We just launched a new version on January 1st, 2013. There's a lot more we want to do so signup today (for free) and you can enjoy the ride along with us!

About Delano

I built Stella to scratch an itch. That is, a web monitoring service that met my needs as a web developer. I've worked on sites large and small and I'm putting everything I learned into building great software for others.

I launched the service in 2010 and relaunched on January 1st, 2013. I've had a lot of fun and learned a lot along the way. I'm using this blog to write about my experiences running the company and the technology behind it.
